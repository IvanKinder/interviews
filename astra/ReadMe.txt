Описание:

    Приложение позволяет создавать, редактировать, удалять задачи;
    Определять задачам срок выполнения, помечать выполненные задачи;
    История изменения задач сохраняется в файл log.txt в корне проекта;
    Есть возможность создавать, редактировать категории, теги и привязывать их к задачам;
    Загружать задачи из xlsx файлов в корне проекта, выгружить задачи из базы данных в файл в корень проекта;
    Примеры xlsx файлов выгрузки / загрузки в корне проекта;

---------------------------------------------------------------------------------------------------------

Endpoints:

    "http://127.0.0.1:8000/api/category/" - категории (get, post)
    "http://127.0.0.1:8000/api/tag/" - теги (get, post)
    "http://127.0.0.1:8000/api/task/" - задачи (get, post)
    "http://127.0.0.1:8000/api/categorytask/" - задавать связи категорий и задач (get, post)
    "http://127.0.0.1:8000/api/tagtask/" - задавать связи тегов и задач (get, post)

    "http://127.0.0.1:8000/api/task/id/" - конкретная задача, доступ по id (get, post, put)
    "http://127.0.0.1:8000/api/export/id/" - экспорт конкретной задачи по id в корень проекта (get)
    "http://127.0.0.1:8000/api/import/filename/" - импорт файла с именем "filename" из корня проекта
    "http://127.0.0.1:8000/schema/" - url для загрузки OpenApi 3 документации

---------------------------------------------------------------------------------------------------------

Инструкция по запуску:

    Распаковать архив,
    Создать виртуальное окружение,
    Установить модули из requirements.txt
    В корне проекта выполнить команду "python manage.py runserver" или "python3 manage.py runserver" в зависимости от системы

---------------------------------------------------------------------------------------------------------
