Инструкция для запуска локально:

    - создать виртуальное окружение при необходимости
    - pip3 install requirements.txt
    - python3 manage.py runserver

Редактирование и удаление категорий возможно только из учетной записи админа (не только из админки):

    username: admin
    password: admin

Второй пользователь для теста:

    username: Ivan
    password: babay12345

В проекте имитация обратной связи с получением данных из полей
